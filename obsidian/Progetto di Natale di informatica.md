# Consegna
Creare un video game da terminale, senza uso di librerie esterne o senza appoggiarsi a framework di nessun tipo. Usare solo le nozioni studiate in classe. Eseguire il lavoro in coppia.

**Gioco scelto**:  Monopoli 
# Elementi necessari
- [ ] Funzione `Tabellone` che mostra il tabellone e che aggiorna la posizione del giocatore 
- [ ] Funzione `Acquista` per comprare la proprietà su cui ci si trova
- [ ] Funzione per recuperare la posizione del giocatore (?)
- [ ] Variabile conto
- [ ] Array per le proprietà ( Gli indici sono il numero delle caselle)
- [ ] Array con i prezzi delle proprietà ( Usando sempre l'indice per accoppiare prezzo-proprietà)
- [x] Funzione TiraDado per muoversi e per gli imprevisti
- [ ] Funzione Probabilità 
- [ ] Funzione Imprevisti
- [x] Funzione show titolo 
- [ ] Funzione regole del gioco

# Obbiettivo
Raggiungere un certo livello monetario, magari con selezione di difficolta,  gestendo i rischi degli imprevisti e gli "attacchi" della banca
